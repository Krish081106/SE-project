# SE-project
# 🚀 Nexventory – Smart Inventory Management System

## 📌 Problem Statement

Small grocery shop owners often struggle with **manual inventory tracking, unexpected stock shortages, and lack of data insights**. This leads to lost sales, overstocking, and inefficient daily operations.

## 💡 Solution

**Nexventory** is a smart inventory management system designed to automate stock tracking, provide real-time insights, and alert shopkeepers when inventory runs low — helping them make better business decisions.

---

## ✨ Features

* 🔔 **Smart Low-Stock Alerts**
  Continuous beep notification every 2 minutes until resolved

* 📊 **Real-Time Dashboard**
  Displays stock, revenue, profit, and customer insights

* 📈 **Sales Reports**
  Weekly & monthly reports for better decision-making

* 🧾 **Bill Generation System**
  Generate and manage customer bills easily

* 📦 **Category-wise Inventory Management**
  Organized stock visualization

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Krish081106/SE-project
cd nexventory
```

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

### 4️⃣ Environment Variables

Create a `.env` file in backend folder:

PORT=3000

MONGO_URI=your_mongodb_connection_string
